## Get started in minutes

<Steps>
  <Step title="Install Clicky">
    Add Clicky to your project via a script tag or your package manager of choice.

    ```html
    <script src="https://cdn.clicky.io/clicky.min.js"></script>
    ```
  </Step>
  <Step title="Initialize the library">
    Call `clicky.init()` with your site ID to start tracking.

    ```javascript
    clicky.init({ siteId: "YOUR_SITE_ID" });
    ```
  </Step>
  <Step title="Track your first event">
    Use `clicky.track()` to capture any custom interaction.

    ```javascript
    clicky.track("button_clicked", { label: "Sign Up" });
    ```
  </Step>
  <Step title="Identify your users">
    Link events to specific users with `clicky.identify()`.

    ```javascript
    clicky.identify("user_123", { name: "Jane Doe", plan: "pro" });
    ```
  </Step>
</Steps>
